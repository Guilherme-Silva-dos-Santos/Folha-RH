# 💼 Sistema de Folha de Pagamento (RH)

Projeto desenvolvido em Java com foco na aplicação de conceitos de Programação Orientada a Objetos (POO).

## 🚀 Funcionalidades

- Cadastro de funcionários (até 10)
- Suporte para dois tipos:
  - Assalariado (salário fixo)
  - Horista (pagamento por horas)
- Cálculo automático de pagamento
- Aplicação de aumento percentual para todos os funcionários
- Exibição dos dados e salários antes e depois do aumento

## 🧠 Conceitos aplicados

- Herança
- Polimorfismo
- Encapsulamento
- Classe Abstrata
- Listas (ArrayList)

## 🛠️ Tecnologias utilizadas

- Java
- NetBeans
- Scanner (entrada de dados)

## 📂 Estrutura do projeto

- `Funcionario` (classe abstrata)
- `Assalariado` (herda de Funcionario)
- `Horista` (herda de Funcionario)
- `Principal` (classe main)
